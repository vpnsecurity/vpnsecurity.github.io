一、Quantumult简介与基本概念

什么是Quantumult？

Quantumult是一款面向iOS平台的网络代理软件，支持多协议代理服务，帮助用户突破网络限制，保护上网隐私。其强大的自定义参数设定和丰富的功能模块，使其不仅适合普通用户，也满足高级用户的个性化需求。

基础概念解析

  • 代理（Proxy）：通过代理服务器转发网络请求，隐藏用户真实IP地址，绕过网络限制。

  • 规则（Rules）：用于判断哪些网络流量走代理，哪些流量直连，提升访问效率。

  • 配置文件（Configuration File）：保存代理服务器、规则、脚本等全部设定，是Quantumult运行的核心。

理解这几个基础概念，有助于您更好地掌握Quantumult的配置技巧。

三、Quantumult的基本参数设定

1. 代理服务器添加与管理

Quantumult支持多种代理类型，包括HTTP、HTTPS、SOCKS5等，用户可以灵活添加：

  • 路径：进入“设置” → “代理” → “添加代理”。

  • 配置代理信息：填写代理服务器地址、端口号、用户名和密码（如需）。

  • 选择代理类型：根据代理服务选择对应协议。

添加完成后，代理列表会自动更新，方便管理和切换。

2. 规则的设置与优化

规则决定哪些流量通过代理，哪些流量直连：

  • 添加规则：在“设置” → “规则”中添加自定义规则。

  • 规则格式：支持域名匹配、IP段匹配等多种格式。

  • 优先级管理：规则顺序影响匹配效果，优先级高的规则优先应用。

合理配置规则，既能保障访问速度，又能确保关键流量通过代理。

3. 配置文件的导入

配置文件集中保存所有代理和规则信息，导入方式多样：

  • 本地导入：在“设置” → “配置文件” → “导入”中选择本地文件。

  • 在线导入：通过配置文件URL直接导入，方便快捷。

导入后，Quantumult自动应用配置，极大简化设置流程。

FAQ

Sing-box 是什么？
Sing-box 是新一代多协议代理框架，支持 VMess、VLess、Trojan、Hysteria、ShadowTLS 等协议。它比传统 V2Ray 更轻量化、模块化，并针对移动端和 Linux 系统进行了性能优化，受到开发者和高级用户的欢迎。
Clash 的策略组可设置延迟阈值吗？
可以。用户可定义延迟阈值，当节点延迟超过设定值时，策略组会自动切换至其他低延迟节点，提高网络速度和稳定性，优化访问体验。
PassWall2 的“节点切换策略”如何配置？
在 PassWall2 中可设置多节点策略，如“按延迟排序”、“自动选择最快节点”等。程序会定期测速，根据实时数据自动切换线路，保证连接的稳定性和速度。
如何给 Clash 或 Sing-box 添加本地广告屏蔽规则？
可导入公共广告域名列表（如 ads、trackers）并将其指向 `REJECT` 或 `DIRECT`（根据需求）；还可以使用 rule-provider 自动更新广告规则，并结合 hosts 或 DNS 层面进行联合拦截以提升屏蔽效果。
VLESS 是否比 Trojan 更稳定？
两者都非常稳定，但 VLESS 在多平台兼容性上略优，而 Trojan 在 HTTPS 伪装和 TLS 安全性上更强。选择取决于服务器环境与具体需求，两者皆为高性能协议。
Quantumult X 中自定义脚本导致规则异常，该如何恢复？
先停用或删除最近添加的自定义脚本，重启应用并观察网络行为是否恢复。若无法恢复，备份并重置配置为默认，然后逐个添加脚本以定位导致异常的具体脚本并修复。
如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
WinXray 的多节点同时使用会影响性能吗？
同时使用多个节点会占用更多系统资源和带宽，可能对性能有一定影响。通过策略组和分流规则合理安排流量，可以降低资源占用，同时保持网络稳定和速度。
Surfboard 的规则导入失败可能是什么原因？
可能是规则格式不兼容、语法错误或策略组缺失。确保导入文件符合 Surfboard 的 YAML 规范，并检查节点或策略组是否完整，可解决导入失败问题。
VMess 连接延迟过高该如何优化？
可尝试切换传输协议（如 WS、gRPC）、调整 MTU 或启用 MUX 多路复用功能。同时检查服务器地理位置、带宽和延迟，并选择最优节点以降低延迟。
