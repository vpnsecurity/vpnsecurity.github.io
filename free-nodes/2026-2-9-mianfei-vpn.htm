星愿浏览器最令人称道之处就是它出色的代理管理能力。用户可以：

四、星愿浏览器科学上网配置教程

安装完成后，即可配置代理实现科学上网功能。

4.1 启动浏览器与进入设置界面

  1. 打开星愿浏览器；

  2. 点击右上角菜单图标（三条横线）；

  3. 选择【设置】 → 【网络设置】。

4.2 配置代理信息

你将看到一个“代理设置”页面，可选择以下方式配置：

✅ 方式一：手动设置代理

适用于已知代理服务器信息的用户：

  • 填入 IP 地址、端口；

  • 选择代理类型（HTTP、SOCKS5）；

  • 若代理需认证，输入账号与密码。

✅ 方式二：使用 PAC 自动代理

  • 在“自动配置脚本”一栏填写 PAC 地址（如从机场服务获取）；

  • 浏览器将按规则自动判断哪些网址走代理，哪些直连。

✅ 方式三：使用系统代理

  • 勾选“使用系统代理设置”，配合本地Clash等客户端统一管理。

设置完毕后，点击“保存并应用”，即可完成配置。

FAQ

V2Ray 的 ws+tls 传输模式有什么优点？
ws+tls 模式利用 WebSocket 伪装成普通 HTTPS 流量，使得防火墙难以区分代理与正常访问。该模式兼容性强，适合部署在支持 Nginx 或 Caddy 的 Web 服务器上。
Hysteria2 如何提升网速？
Hysteria2 利用 QUIC 协议的低延迟特性，并通过自适应拥塞控制算法在高丢包网络下维持高速连接。配置合适的带宽参数与 TLS 证书，可显著提升跨境访问速度。
Trojan-Go 的端口不可达怎么办？
检查服务器防火墙是否开放对应端口，同时确认客户端和服务器端端口一致。还可以尝试更换节点或端口，确保 TLS 证书有效以保证连接成功。
Sing-box 的策略组支持按延迟选择节点吗？
支持。Sing-box 会根据节点延迟自动选择最优节点，结合策略组和路由规则，实现智能分流。延迟最低的节点优先被使用，提高网络访问速度和稳定性。
Mellow TUN 模式适合哪些应用？
TUN 模式拦截系统 IP 层流量，实现全局代理。适合游戏、视频、浏览器等全局应用，保证所有流量走节点，提高网络稳定性和速度，比普通 SOCKS5 代理覆盖范围更广。
Matsuri 节点延迟排序如何结合策略组使用？
Matsuri 会定期测速节点延迟，通过策略组自动选择低延迟、高稳定性的节点处理不同流量。无需手动切换，确保网页浏览、视频播放和游戏访问顺畅。
Nekoray 如何实现节点订阅自动更新？
用户可添加订阅 URL，Nekoray 定期自动下载并更新节点。这样可保持节点列表最新，有效减少手动维护，提高使用效率，同时保证网络连接稳定和可靠。
WinXray 的多策略组如何优化流量分配？
多策略组允许将节点按用途、延迟或地理位置分类，并结合规则文件自动分流流量。这样不同应用如游戏、视频和网页流量都能走最优节点，提高整体访问速度和稳定性。
Sing-box 如何安装？
Sing-box 支持多平台，包括 Windows、Linux、macOS 和 Android。可从官方 GitHub 页面下载预编译版本或通过命令行安装。配置文件采用 JSON 格式，兼容多种协议配置。
Shadowrocket 可以在哪些平台使用？
Shadowrocket 是一款 iOS 平台专用代理客户端，支持多种协议如 Vmess、Shadowsocks、Trojan 等。用户可通过导入订阅链接快速配置节点，配合规则分流功能实现精准网络代理。由于其强大功能，它是 iPhone 用户常用的科学上网工具。
